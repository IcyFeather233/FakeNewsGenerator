# FakeNewsGenerator
SCU社团联新闻稿生成器

## 说明

项目采用 Django + Vue + ElementUI。

appFront 是前端部分，其他部分是后端部分。

运行后访问 `http://127.0.0.1:8000/fakenews/#/` 即可。

## 如何运行

```
# 进入venv
. ./bin/activate

# 运行
python manage.py runserver [port]
```
